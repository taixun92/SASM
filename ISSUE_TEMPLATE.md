## 설명
정수 덧셈기를 만들어 봤습니다.

## 재현 방법
<!-- 이슈를 재현하는 방법을 단계별로 설명하세요. -->
1. 실행 시 2개의 정수 인자를 받는다.
2. 두 인자를 더한다.
3. 결과를 출력한다.

## 실행 방법
```bash
> python add.py -a 1 -b 2
```
## 예상 동작
```bash
> The answer is [ 1 + 2 = 3 ]
```

## 실제 동작
```bash
> The answer is [ 1 + 2 = -1 ]
```
